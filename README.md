# Student Management System

This repository contains the full source code for the Student Management System, split into frontend and backend.

## Structure

*   **[frontend](./frontend)**: The React-based user interface.
*   **[backend](./backend)**: The Spring Boot server-side application.

## getting Started

### Frontend
Navigate to the `frontend` directory:
```bash
cd frontend
npm start
```

### Backend
Navigate to the `backend` directory:
```bash
cd backend
./mvnw spring-boot:run
```
